# PHP Backend Tasks API

API REST construida en PHP puro sin frameworks, diseñada para demostrar habilidades backend reales.

## Características

- Arquitectura MVC ligera
- Router manual
- API RESTful
- Respuestas en JSON
- CRUD completo de tareas
- Soft Delete (`deleted_at`)
- Restauración de tareas eliminadas

## Estructura del proyecto

- controllers/
- models/
- routes/
- config/
- core/
- public/

## Cómo ejecutar

```bash
php -S localhost:8000 -t public
